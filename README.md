## 项目描述

### 项目选题

油炬智愿 —— 中国石油大学（北京）克拉玛依校区志愿服务平台

### 技术栈

- 后端：Java SpringBoot, Spring MVC, Spring Security, Mybatis-Plus,JWT
- 构建工具：Maven（后端）,Vite（前端）
- 前端： Vue3、Pinia、Vue Router、Axios、Element Plus
- 数据库：MySQL
- 版本管理：Git
- 代码托管：GitHub
- 服务器：Docker

### 开发环境

- 后端开发环境: JDK 21 (LTS)
- 前端开发环境: Node.js 22 (LTS)
- 容器化环境: Docker Desktop
